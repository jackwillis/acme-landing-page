# Acme Corp Landing Page

Marketing landing page for Acme Corp — anvil logistics and supply chain management. Single-page site, no backend.

## Tech Stack

- Single file: `index.html`
- Tailwind CSS via CDN
- Vanilla JS only — no frameworks, no build step
- Must work by opening the HTML file directly in a browser

## Brand

- **Colors:** Burnt umber primary (`#8B4513`), amber accent (`#F59E0B`), dark gray text
- **Font:** Inter via Google Fonts
- **Logo:** `assets/acme-logo.svg`
- **Tone:** Professional but friendly. Think "competent startup," not "enterprise sales deck."

Custom umber palette is defined in the Tailwind config in `index.html`.

## Constraints

- Responsive, mobile-first
- Semantic HTML, sufficient contrast, alt text
- No placeholder images — use SVG, gradients, or emoji
- Don't create additional files
- Don't add npm, package.json, or any build tooling
